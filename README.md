# WebFonts

블로그에서 쓰려고 모아두는 웹폰트 저장소. JSDelivr를 이용하여 CDN 서비스를 제공한다.


## FontFamilies
`koPub Dotum`
`KoddiUDOnGothic`
## FontWeights
`300` : Light
`500` : Medium
`700` : Bold
`900`: ExtraBold


## How to use
### HTML
```html
<link rel="stylesheet" href="//cdn.jsdelivr.net/gh/CXZ7720/webbfonts@main/kopubdotum.css">
```

### CSS
```css
@import "//cdn.jsdelivr.net/gh/CXZ7720/webfonts@main/kopubdotum.css";
```
