# kopub-world

한국출판인회의에서 제공하는 KoPubWorld 돋움체 웹폰트

## FontFamilies
`koPub Dotum`

## FontWeights
`300` : Light
`500` : Medium
`700` : Bold


## How to use
### HTML
```html
<link rel="stylesheet" href="//cdn.jsdelivr.net/gh/CXZ7720/kopub-world@1.0/kopubdotum.min.css">
```

### CSS
```css
@import "//cdn.jsdelivr.net/gh/CXZ7720/kopub-world@1.0/kopubdotum.min.css";
```
